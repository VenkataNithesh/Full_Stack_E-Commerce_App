# S Cart (E-Commerce) Project

## Presentation on E-Commerce (Online Shopping)

---

## What is E-Commerce?

E-Commerce, commonly known as an online marketplace, consists of buying and selling goods and services through electronic systems such as the Internet and other computer networks.

---

## Project Overview

S Cart is an e-commerce web application that enables users to browse, select, and purchase products online.

### Key Functionalities

- Browse available products
- View detailed product information
- Add products to a shopping cart
- Complete checkout by providing:
  - Name
  - Email
  - Address
- Cart management using Browser Local Storage
- Secure admin authentication
- Admin panel for product management
- Database storage for products and orders

### Admin Features

- Admin Login
- Add Products
- Edit Products
- Delete Products
- Manage Product Catalog
